# Evidence-Fused Neutrosophic Framework

Official implementation of the Evidence-Fused Neutrosophic Framework, a hybrid multi-criteria decision-making (MCDM) model for uncertainty-aware treatment selection.

## **Overview**
The framework integrates Neutrosophic Logic, Interval-Valued Fuzzy Sets (IVFS), and Dempster-Shafer Theory (DST) to provide robust and discriminative rankings in complex medical decision scenarios.

### **Included Contents:**
*   `/4_Code`: Main framework script including Monte Carlo simulation and visualization logic.
*   `/4_Code/figures/png`: High-definition visualizations as requested by reviewers.
